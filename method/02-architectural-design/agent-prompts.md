# Agent Prompts - Architectural Design

## Iteration Plan

```text
Con los drivers aprobados y el modelo de dominio, crea iteration-plan.md. La primera iteracion estructura el sistema; las siguientes atienden drivers de mayor prioridad. Cada fila debe tener numero, meta, drivers y elementos probables a refinar.
```

## ADD Step

```text
Ejecuta la iteracion ADD indicada en iteration-plan.md siguiendo method/02-architectural-design/ADD.md. Ejecuta un solo paso por respuesta y espera aprobacion del arquitecto antes de avanzar al siguiente paso.
```

## Tradeoffs

```text
Para los drivers seleccionados, propone maximo 3 conceptos de diseno. Compara pros, contras, alternativas descartadas y consecuencias. No inventes requisitos ni tecnologias fuera del contexto aprobado.
```
